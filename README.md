# Proyecto Semestral

Objetivo: Dado un dataset público se debe realizar un proyecto de minería de datos. Para este caso se trabajó con datos de Twitter para realizar un análisis de sentimientos durante la pandemia. Para ello se usaron los siguientes datos: [COVID19-Tweets](https://www.kaggle.com/datasets/gpreda/covid19-tweets)

Se trabajó en base a tres hitos acumulativos. Los cuales siguieron la siguiente estructura:
1. Motivacion:
2. Descripción de base de datos
3. Exploración de datos
4. Mejora Hito 1
5. Propuesta experimental
6. Referencias
7. Contribuciones del equipo
